
Manager website bookmark.
Usage:
    bookmark <action> [--option] [<value>]...

    subcommands:
        help           Print help documents.
        version        Get bookmark version.
        page           Operation of website.
        folder         Operation of page folder.

    page
        add             Add a new page.
        drop            Drop a page.
        edit            Edit a page.
        list            Get a list of page.
        describe        Get page details.
        open            Open a page.

        1. Add a new page.
        bookmark page add <name> --url <url>

        2. Drop a page.
        bookmark page drop <name>

        3. Edit a page
        bookmark page edit <name> [--name <new_name>] [--url <url>]

        4. Get a list of page
        bookmark page list

        5. Get page details
        bookmark page describe <name>

        6. Open a page
        bookmark page open <name>

        e.g.
        bookmark page add baidu --url http://www.baidu.com/

    folder:
        add        Add a folder.
        drop       Drop a folder.
        edit       Edit a folder.
        list       Get a list of folder.
        desc       Get folder details.
        use        Use a folder.
        current    Get current use folder.
        resize     Resize a folder.

        1. Add a folder
        bookmark folder add <name> --desc <describe>

        2. Drop a folder
        bookmark folder drop <name>

        3. Edit a folder
        bookmark folder edit <name> --name <new_name> --desc <new_desc>

        4. Use an another folder
        bookmark folder use <name>

        5. Get current folder
        bookmark folder current

        6. Resize folder
        bookmark folder resize
